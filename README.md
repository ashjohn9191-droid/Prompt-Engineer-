# 🚀 Prompt Engineer Toolkit

A collection of Python scripts demonstrating different **Prompt Engineering techniques** for working with AI/LLMs. This repository is designed to help beginners and intermediate developers understand how to structure, optimize, and experiment with prompts.

---

## 📂 Project Structure

```
Prompt-Engineer/
│── Adv_PE.py                  # Advanced prompt engineering examples
│── app.py                     # Main application entry point
│── client.py                  # Client-side interaction with model/API
│── code4.py                   # Additional experimentation code
│── def_input_prompt.py        # Prompt with default input handling
│── list_strings_prompt.py     # Prompt handling lists of strings
│── loop_prompt.py             # Prompt generation using loops
│── prompt_templates_f_strings.py  # Prompt templates using f-strings
│── simple_prompt.py           # Basic prompt examples
│── README.md                  # Project documentation
```

---

## ✨ Features

- 🧠 Basic to advanced prompt engineering examples  
- 🔁 Dynamic prompt generation using loops  
- 🧩 Template-based prompt design using f-strings  
- 📋 Handling structured inputs like lists  
- ⚡ Clean and modular Python scripts

## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/Prompt-Engineer.git
cd Prompt-Engineer
```

2. (Optional) Create virtual environment:
```bash
python -m venv venv
```

Activate environment:

- Windows:
```bash
venv\Scripts\activate
```

- Mac/Linux:
```bash
source venv/bin/activate
```

3. Install dependencies (if any):
```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run any script individually:

```bash
python simple_prompt.py
```

or

```bash
python app.py
```
